```
[section]-[topic]-[number].md

- overview-getting-started-01.md

- example-basic-workflow-01.md
- best-practices-performance-01.md
```

All replaceable content wrapped in **(parentheses)**:

- **(System Name)** - Your platform/product name
- **(Organization Name)** - Your company name
- **(Industry)** - Industry vertical (Finance, Healthcare, etc.)
- **(Job Title)** - Role using the system
- **(Region)** - Geographic region (US-East, EU, APAC)
- **(Version)** - Software version numbers
- **(Color)** - Brand color codes
- **(Team Name)** - Team responsible for component
- **(Environment)** - Dev, Staging, Production
- **(Tool/Service)** - External tool or service name

**Example**:

> (System Name) is an enterprise-grade platform for (Industry) professionals at (Organization Name), deployed across (Region) with 99.99% uptime SLA.
